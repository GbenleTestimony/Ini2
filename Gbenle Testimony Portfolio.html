About Me

Hi!, I'm Testimony - a curious blend of medicine and technology. Currently a medical student at the University of Ilorin, I'm deeply passionate about the future of healthcare and how technology can bridge gaps in patient care, especially through telemedicine. Alongside ny medical training, I've been learning software development, focusing on web technologies and Python programming
Whether it's diagnosing a condition or debugging a function, I'm committed to solving problems that matter - and I believe technology is one of the most powerful tools in a future doctor's hands.

So Far.    ;    What Next?

Here are some of the core technologies I'm learning and building with as well as other branches to be explored:

HTML

The skeleton of the web. I use HTML to structure content meaningfully and semantically. From creating clean layouts to integrating forms and media, it's the starting page of every page I build.

CSS

CSS helps me style pages to look just right. I enjoy exploring modern layout techniques like Flexbox and Grid, and I'm learning how to make my designs responsive, clean and user-friendly. A good UI isn't just pretty - it's purposeful

Python

Python is the first language that really made sense to me. It's clean syntax and wide range of applications - from automation to AI - make it an essential tool in both medical data analysis and software development. I'm especially drawn to its use in healthcare analytics and backend services

JavaScript

JavaScript brings life to my web pages. I'm exploring how to build interactive components, validate forms, and eventually create dynamic web applications . It's a steep but exciting learning curve and I'm here for it.

My Interest in Telemedicine

In a country where access to quality healthcare is still uneven, telemedicine holds the power to transform lives. I'm passionate about building or contributing to tools that bring medival help closer to people who need it - especially in rural or underserved arears.
As I grow in both medicine and software engineering, I hope to merge these paths to create meaningful health - tech solutions that work in real - world contexts.
